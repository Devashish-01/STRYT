# Booking an Appointment

Already know exactly who you want? Book directly instead of going through the Ask/Quote system.

**How to book:**
1. Open any shop or provider's profile and tap **"Book appointment"**.
2. Pick a service/package, then a date and time slot from their real, current availability.
3. Add notes or a reference photo if it helps them prepare.
4. Confirm. Some sellers require payment upfront before they'll accept; others let you pay around the time of service — this is set by them, and you'll see which applies before you confirm.

**Shopping cart.** For a shop with a catalog (a menu, a product list), you can add multiple items to a running cart from the Menu/Catalog tab, then check out by picking a single pickup/collection time slot — no need to book each item separately.

**What happens after you book.** The seller can **accept or decline** your request (with a note if they decline). A booking is only truly confirmed once they accept, unless their settings auto-confirm. If your slot isn't accepted quickly, message them directly.

Manage everything you've booked from **My Appointments** — see that guide for cancelling, rescheduling, and rebooking.
