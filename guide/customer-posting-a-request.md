# Posting a Request ("Ask Your Street")

Can't find exactly what you need by browsing? Post a request and let nearby shops, providers, and neighbours send you real quotes.

**How to post:**
1. Tap **"Need something? Ask your street"** on Home, or open the **Requests** tab.
2. Pick a quick-start template if one fits (Birthday cake, Plumber, AC service, Daily tiffin, etc.) — it pre-fills smart fields for you — or start from scratch. You can dictate your description with voice-to-text.
3. Add up to 4 photos, a budget range (fixed or hourly), and when you need it by.
4. Choose how long the request stays open (up to 24 hours) and how far around you it should be shown.
5. Optional switches: mark it **urgent**, **recurring**, or post **anonymously** (your identity stays hidden from other users, though STRYT can still see it internally for safety/moderation).

**Group buy.** You can mark a request as a group buy if you're hoping other neighbours want the same thing too, to try to unlock bulk pricing from a responder. *(The "join this group buy" tap is being reworked right now — for now, treat group-buy requests the same as a regular request; a responder can still see the interest and quote accordingly.)*

Once posted, nearby shops/providers/neighbours can respond with a priced quote — see **"Quotes & Deals"** for what happens next.
